# Iris Logo Spec

## Current Assets

- `assets/logo/generated/iris-mark-transparent.png`
- `assets/logo/generated/iris-mark-white.png`
- `skills/iris/assets/iris-mark-white.png`

The transparent raster PNG is the primary mark for README and rendered HTML. The white PNG is used by plugin and skill surfaces that need a light mark on a colored background. There is no current editable SVG source in this repository.

## Concept

The Iris mark is a quiet botanical line drawing of a side-opening iris. It supports the product metaphor: analysis brought into focus and composed into a readable report.

## Usage

Use the mark sparingly. Iris reports should be identified by the report title, thesis, and findings before decorative branding.

Recommended minimums:

- small chip: `32 px`
- composer icon: `48 px`
- documentation header: `96 px`

## Misuse

Do not:

- add gradients to the mark
- fill the petals with saturated color
- add a pupil, eye, or literal vision symbol
- redraw the flower as a realistic iris
- add shadows or bevels
- rotate the mark
- use more than one accent color inside the mark
